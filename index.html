RPG Core Framework

Built a modular RPG engine from scratch. It features a secure DataStore system with JSON-encoded attributes, a centralized NPC spawner with basic AI, and a weighted loot table system. I also handled the full-stack side, creating a hotbar UI that communicates with the server for item usage and a dynamic quest tracker.

Grid-Based Puzzle Game (Block Blast)

Created a 2D puzzle system on a 3D grid. The logic handles part-snapping, collision detection to prevent overlapping pieces, and an efficient line-clearing algorithm. It includes a scoring system that triggers whenever a row or column is fully occupied.

Procedural Medieval Map Generation

Scripted a procedural generation system that creates unique medieval layouts. It uses coordinate-based math to place pre-modeled assets dynamically, ensuring the map feels varied while maintaining a consistent theme and performance-friendly part count.

External Data RNG Game

An experimental RNG project that bypasses traditional DataStores. I used HttpService to fetch and store player data directly on a GitHub repository. This project demonstrates my ability to work with external APIs and handle data serialization outside of the Roblox environment.

Trivia Word Game (JKLM PopSauce Inspired)

A real-time trivia game where players compete to answer prompts. To keep the content easily updatable without restarting servers, I hosted the entire question database on GitHub and pulled the data live into the game via HTTP requests.

Memory Card Game (Full Project)

A complete, polished project featuring card-flipping logic and match-detection algorithms. Beyond the core game, I built a full ecosystem including an in-game shop, currency system, and custom character "chair" skins to demonstrate a finished, monetization-ready product.

Incremental / Clicker Game

Developed a scalable "Weight" system for an incremental game. I focused on making the math efficient to handle large numbers and implemented a prestige mechanic. The system is designed to easily add new upgrades and multipliers through a centralized configuration module.
